## LAS 4 MÉTRICAS QUE MANDAN

### 1. SUELDO DEL FUNDADOR

**La métrica más ignorada y más importante.**

- Si EKIO factura pero Javier no se paga un sueldo digno, el juego no está ganado
- El sueldo da estabilidad mental para ejecutar con claridad
- Las decisiones desde el miedo económico siempre son peores

**Objetivo**: Definir el sueldo mensual que Javier necesita para vivir bien → este número es el primer objetivo financiero del negocio.

---

### 2. GP/CAC — La métrica que decide si puedes escalar

```
GP/CAC = Gross Profit por pedido / CAC (Coste de Adquisición de Cliente)

REGLA DE ORO:
→ GP/CAC > 2: PUEDES ESCALAR adquisición
→ GP/CAC < 2: NO escales. Arregla primero el margen o el CAC.
→ GP/CAC < 1: Estás perdiendo dinero en cada cliente nuevo
```

**Por qué GP/CAC > 2:**
Con GP/CAC = 2 tienes margen para pagar:
- Envíos
- Equipo / operaciones
- Software (Klaviyo, Shopify, etc.)
- Impuestos
- Y aún quedarte con beneficio neto

**Cómo calcularlo para EKIO:**
```
Ejemplo Spiro Disc:
  Precio venta:           49€
  COGS (coste producto):  -12€
  Envío:                  -5€
  Stripe fees (2,9%+0,30): -1,72€
  ─────────────────────────────
  Gross Profit:           30,28€

  CAC (lo que gastas en ads para conseguir 1 cliente): 15€

  GP/CAC = 30,28 / 15 = 2,02 ✅ (justo en el límite, hay que mejorar)
```

### 3. CONTRIBUTION MARGIN (CM) — La verdad financiera por producto

```
CM = Revenue - COGS - Envío - Fees de pago - Marketing variable

CM positivo en 1ª compra → Puedes escalar con confianza
CM negativo en 1ª compra → Solo viable si LTV es suficientemente alto
```

### 4. MER — Marketing Efficiency Ratio

```
MER = Revenue Total / Gasto Total en Marketing

Gasto total incluye TODO:
  Ads (Meta, Google, TikTok)
  + Klaviyo / software email
  + Agencia (si aplica)
  + Producción de contenido
  + Influencers
  + Herramientas (Hotjar, etc.)

MER objetivo EKIO: > 3
MER > 3: Buena forma
MER < 2: Hay problemas de rentabilidad
```

**Por qué MER > ROAS aislado:**
```
ROAS de Meta puede ser 4x (parece bien)
Pero si incluyes todos los costes de marketing: MER = 1,8 (en realidad mal)

MER es la métrica holística. El ROAS de un canal es solo una parte del puzzle.
```

## SISTEMA DE BENEFICIO — JESÚS OROZCO

### La máquina de beneficios crecientes:

```
PASO 1: Adquisición rentable (GP/CAC > 2)
    ↓
PASO 2: Base de clientes activos que crece
    ↓
PASO 3: Sistema de recurrencia (12 campañas/año + flujos)
    ↓
RESULTADO: Beneficio neto predecible y creciente
```

### El efecto dominó de la base activa:
```
1.000 clientes activos × 3% recompra/mes = 30 pedidos/mes (sin pagar CAC)
3.000 clientes activos × 3% = 90 pedidos/mes
10.000 clientes activos × 3% = 300 pedidos/mes

Estos pedidos tienen CM altísimo porque no pagas adquisición.
→ La base activa es el activo más valioso de EKIO.
→ Como un bloque de pisos en alquiler: cada mes genera rentas.
```

## LAS 3 PALANCAS DE MARGEN

### Palanca 1: Subir precios inteligentemente

No todos los productos deben subir precio. Identifica:
- Productos donde el cliente tiene baja sensibilidad al precio (paneles de alto ticket)
- Productos donde EKIO tiene diferenciación clara vs. competencia
- Productos donde el cliente percibe alto valor pero el precio está bajo de mercado

Test de precio: Sube 10-15% en 1 producto. Si la conversión baja < 10%, el CM neto mejora.

### Palanca 2: Simplificar catálogo

Vende y promociona PRIMERO lo que más margen deja.
Si el producto A tiene CM de 40% y el B de 15%, cada euro de marketing va al A.

**Regla práctica**: Si un producto tiene CM < 20%, necesita justificación especial (volumen, gateway a upsell, etc.) o se retira.

### Palanca 3: Dejar de competir en precio

Competir en precio = carrera hacia el fondo.
EKIO compite en: conexión emocional + autoridad científica + propuesta clara.
El cliente que compra por precio no es el cliente de EKIO.
