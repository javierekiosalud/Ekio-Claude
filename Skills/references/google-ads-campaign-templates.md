```

    □ Producto en Merchant Center aprobado
    □ URL de landing funcionando con UTMs
    □ Enhanced Conversions verificando email
    □ RSA creado con 15 headlines
    □ Asset group PMax creado con imágenes y vídeo

SEMANA 2 — RAMP UP:
  Campaña 1: Shopping Standard (60% del budget)
    Estrategia: Maximize Conv. Value
    Keywords: brand + producto específico exact match
    Custom label: "nuevo_producto" (prioridad máxima)

  Campaña 2: Search Non-Brand (40% del budget)
    Keywords: exactas del producto + long tail
    RSA: versión específica del producto lanzado
    Bid: Manual CPC alto inicialmente para ganar datos rápido
    → Cambiar a Smart Bidding cuando ≥20 conversiones

SEMANA 3-4 — OPTIMIZACIÓN:
  Revisar Search Terms Report → añadir negatives
  Revisar ad strength de RSA → iterar assets "Low"
  Si ≥30 conversiones → activar tROAS
  Ajustar budget hacia campaña con mejor ROAS
  Crear remarketing list de visitantes del producto
  → Activar RLSA en campañas Search con bid modifier +30%

POST-LANZAMIENTO (mes 2+):
  Graduación a PMax asset group del producto
  Añadir custom intent audience del producto al PMax general
  Customer Match update con compradores del nuevo producto
```

## QUICK REFERENCE — COMANDOS DE DIAGNÓSTICO RÁPIDO

```
CUANDO EL ROAS CAE DE GOLPE:
  1. ¿Hay errores en Merchant Center? → Feed roto = Stop Shopping
  2. ¿Consent Mode v2 sigue activo? → Si no, datos perdidos
  3. ¿Alguna campaña en "learning period"? → No tocar, esperar
  4. ¿CPCs subieron? → Ver Auction Insights (competencia nueva)
  5. ¿Landing page cae? → Pingtree o PageSpeed Check

CUANDO EL VOLUMEN CAE SIN QUE CAIGA EL ROAS:
  1. ¿Budget limitado en horas pico? → Subir presupuesto
  2. ¿Impression Share cayó? → Ver Lost IS (budget vs rank)
  3. ¿Feed tiene productos sin aprobar? → MCN diagnóstico
  4. ¿tROAS target demasiado alto? → Bajar ligeramente

CUANDO EL CPC SUBE MUCHO:
  1. Ver Auction Insights → ¿competidor nuevo o más agresivo?
  2. Revisar Quality Score → ¿cayó en keywords principales?
  3. ¿Keywords muy genéricas sin negatives? → Añadir negatives
  4. ¿Es Q4 / temporada alta? → CPCs normalmente suben en noviembre-diciembre

CUANDO EL CTR DE SHOPPING ES BAJO (<0.8%):
  1. Revisar imágenes → ¿Foto atractiva? ¿Fondo limpio?
  2. Revisar títulos → ¿Primeros 70 chars son los mejores?
  3. Revisar precio → ¿Competitivo en búsquedas de comparación?
  4. ¿Hay reseñas de productos? → Stars en Shopping mejoran CTR +5-10%
```
