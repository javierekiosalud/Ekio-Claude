# Skill: SEO On-Page para Shopify

## Optimización de páginas de producto

Para cada producto, genera los siguientes elementos:

```
TITLE TAG (≤60 caracteres):
[Nombre Producto] — [Beneficio Principal] | [Marca]

META DESCRIPTION (≤160 caracteres):
[Beneficio/transformación]. [Dato diferenciador]. [CTA]. Envío gratuito.

H1:
[Nombre Producto]: [Beneficio Principal para el Usuario]

DESCRIPCIÓN SEO (300-600 palabras):
Párrafo 1: Problema que resuelve (con dato científico si aplica)
Párrafo 2: Cómo funciona (beneficios, no características)
Párrafo 3: Diferenciadores (patente, estudios, certificaciones)
Párrafo 4: Para quién es (perfil de usuario ideal)
Párrafo 5: Qué incluye / especificaciones clave
Párrafo 6: Garantía y confianza

BULLET POINTS OPTIMIZADOS (5-7):
✓ [Beneficio] — [Prueba o dato de respaldo]

ALT TEXT PARA IMÁGENES:
[Nombre producto] - [descripción visual] - [keyword relevante]
```

---

## Optimización de páginas de colección

```
TITLE TAG: [Categoría] — [Beneficio Colectivo] | [Marca]
META DESCRIPTION: Descubre [categoría]. [Propuesta de valor]. [Nº productos] opciones.
H1: [Categoría]: [Propuesta de valor]
TEXTO DESCRIPTIVO (150-300 palabras): Contexto + keywords + enlaces internos
```

## Optimización de homepage

```
TITLE TAG: [Marca] — [Propuesta de Valor Principal] | [Categoría]
META DESCRIPTION: [Marca] es [qué hace]. [Diferenciador]. [CTA].
H1: [Propuesta de Valor Principal]
```
